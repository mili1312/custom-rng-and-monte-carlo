# Custom Random Number Generator & Monte Carlo Simulation (MATLAB)

This project implements custom pseudo-random number generators and statistical distribution transformations using MATLAB.

## Implemented Components

- Linear Congruential Generator (LCG)
- Box-Muller transformation (Normal Distribution)
- Triangular distribution via inverse CDF
- Statistical validation tests
- Histogram-based distribution verification

## Key Concepts

- Pseudo-random number generation
- Inverse transform sampling
- Monte Carlo simulation
- Distribution validation
- Statistical testing



## What I Learned

- Implementing RNG algorithms from scratch
- Distribution transformation techniques
- Statistical validation of generated samples
- Monte Carlo based simulation modelling
